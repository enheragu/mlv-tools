# mlv-tools

mlv-tools is a static web landing that unifies access to related statistical tools.

It provides:

- a single entry point for multiple tools,
- a consistent UI shell and navigation,
- language and theme controls,
- direct access to each tool page.

Web version:

- https://enheragu.github.io/mlv-tools/

Documentation:

- i18n policy: `docs/i18n-policy.md`
